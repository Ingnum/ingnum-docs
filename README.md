# INGNUM

#### Ingnum Networksとは

[イングナムネットワークス](https://www.ingnum.net/)(Ingnum Networks)は、Minecraftを始めとしたゲームサーバーに特化したホスティングサービスです。業界最安値でサーバーをご提供させて頂いております。また、ITに興味を持つ学生を対象に様々なインターネット関連事業を展開しております。
